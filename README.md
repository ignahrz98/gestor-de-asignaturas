# Gestor de asignaturas

Es un pequeño programa que permite llevar el control de las asignaturas y los pendientes, para cumplir con los deberes académicos.

El programa utiliza ficheros binarios para almacenar la información. El propio programa los va creando a medida que se vaya utilizando.

Se ejecuta en la linea de comandos. El programa hace uso de una función llamada **gotoxy** para hacer un poco más gráfica la aplicación, y va cambiando los colores de la terminal para hacer más interactivo su uso.

## Conceptos básicos

### El menú

Al iniciar el programa muestra un menú con el que se interactúa ingresando el número correspiente a la opción deseada.

### Unidades curriculares

Para ingresar la información sobre las materias vistas en clase.

### Registrar día y registro diario

Con la primera opción crear una entrada para su pequeño diario y la segunda muestra todo su historial de clases.

### Eventos

Esta opción permite anotar las actividades pendientes, indiciando la prioridad de la actividad para agruparlas según se desee.

**Nota:** *Los Eventos con prioridad alta se muestran al iniciar el programa para siempre tenerlos a la vista. No olvide nada nunca más.*
